# Text Chunking

Large documents must be split into smaller pieces before embedding.

This process is called **chunking**.

---

## Why Chunking Matters

- Models have context limits
- Smaller chunks improve retrieval precision
- Chunk size affects relevance and cost

---

## Common Strategies

- Fixed-size chunks
- Overlapping windows
- Semantic chunking
